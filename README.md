# Real-Time Face Detection Project


## 📌 Project Overview
This project implements a real-time computer vision system to detect human faces using a webcam feed. It utilizes the **Viola-Jones framework**, specifically the **Haar Cascade Classifier**, to identify facial features through a series of rapid calculations.

As part of a Data Analytics program, this script serves as a foundational tool for gathering spatial data (face coordinates) and temporal data (detection frequency) from video streams.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Library:** OpenCV (`opencv-python`)
* **Model:** Pre-trained `haarcascade_frontalface_default.xml`
